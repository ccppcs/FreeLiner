# Free Liner는 MFC로 제작한 2D 스포츠 아케이드 게임 입니다.

   * Free Line은 스케이트의 한 종류 입니다.
   * Free Line을 타는 모습의 케릭터를 조종하며 시간 내에 높은 점수를 획득하는 것이 목표인 게임입니다.
   * q, w 키룰 번갈아 입력하면서 속도를 증가시킬 수 있습니다.
   * 일정 속도를 만족하면 좌측 상단에 표시 되어 있는 스킬을 사용 할 수 있습니다.
   * 장애물을 통과할 시 더 높은 점수를 받을 수 있습니다.
